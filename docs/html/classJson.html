◆ move()

virtual void Json::move ( const string & from,
string & to,
int depth = 0 )
inlinevirtual

Moves an existing JSON element from the first path to the second one.

If the path is invalid the JSON state is not changed. If given invalid 'to' path the element from 'path' will be deleted.

The JSON element at the first path is deleted.

Parameters
fromRecursive path, separated by '/'. The last argument is the existing JSON element's key that will be moved.
toRecursive path, separated by '/'. The last argument is the JSON element's new location. Use './' when trying to move element at the top level of the JSON file.
depthOptional param, not to be used.

Reimplemented in JsonObject.
