Parses a JSON object string and constructs a \mbox{\hyperlink{classJsonObject}{Json\+Object}}. 

This method processes the input string to extract key-\/value pairs and constructs a {\ttfamily \mbox{\hyperlink{classJsonObject}{Json\+Object}}} from them. It uses {\ttfamily parse\+Object\+Key} to extract keys and {\ttfamily \mbox{\hyperlink{namespaceUTILITIES_af1224e12ff9f834b75c0c0ccff4ec3af}{UTILITIES\+::parse\+Value}}} to extract values. The extracted values are parsed into {\ttfamily \mbox{\hyperlink{classJson}{Json}}} objects using the {\ttfamily \mbox{\hyperlink{classJsonFactory}{Json\+Factory}}}. Each parsed key-\/value pair is stored in a vector of {\ttfamily \mbox{\hyperlink{classJsonPair}{Json\+Pair}}} and used to create a new {\ttfamily \mbox{\hyperlink{classJsonObject}{Json\+Object}}}.
