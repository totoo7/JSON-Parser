Parses a JSON object string and constructs a \doxylink{classJsonObject}{Json\+Object}. 

This method processes the input string to extract key-\/value pairs and constructs a {\ttfamily \doxylink{classJsonObject}{Json\+Object}} from them. The extracted values are parsed into {\ttfamily \doxylink{classJson}{Json}} objects using the {\ttfamily \doxylink{classJsonFactory}{Json\+Factory}}. Each parsed key-\/value pair is stored in a vector of {\ttfamily \doxylink{classJsonPair}{Json\+Pair}} and used to create a new {\ttfamily \doxylink{classJsonObject}{Json\+Object}}.

This method extracts the key from a JSON object string starting at the specified index. The key is expected to be enclosed in double quotes. The method handles quoted keys and updates the index to the position after the key-\/value separator (\textquotesingle{}\+:\textquotesingle{}).
