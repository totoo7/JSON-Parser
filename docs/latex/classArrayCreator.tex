Parses a JSON array string and constructs a \mbox{\hyperlink{classJsonArray}{Json\+Array}}. 

This method processes the input string to extract values and constructs a \mbox{\hyperlink{classJsonArray}{Json\+Array}} from them. It uses \mbox{\hyperlink{namespaceUTILITIES_af1224e12ff9f834b75c0c0ccff4ec3af}{UTILITIES\+::parse\+Value}} to extract values from the input string. Each extracted value is parsed into a \mbox{\hyperlink{classJson}{Json}} object using the \mbox{\hyperlink{classJsonFactory}{Json\+Factory}}. The parsed \mbox{\hyperlink{classJson}{Json}} objects are cloned and stored in a vector of Json$\ast$, which is then used to create a new \mbox{\hyperlink{classJsonArray}{Json\+Array}}. The temporary {\ttfamily \mbox{\hyperlink{classJson}{Json}}} objects are deleted after cloning to manage memory.
