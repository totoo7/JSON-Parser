\doxysubsection{Detailed Description}
Parses a JSON value from a string. 

This function extracts a single JSON value from the given string, starting at the position indicated by the {\ttfamily index} parameter. It handles JSON objects, arrays, strings, numbers, booleans, and null values. The function updates the {\ttfamily index} to point to the next character after the parsed value.


\begin{DoxyParams}{Parameters}
{\em value} & The JSON string from which to parse the value. \\
\hline
{\em index} & A reference to the current position in the string. This is updated as the function parses the value. \\
\hline
\end{DoxyParams}
